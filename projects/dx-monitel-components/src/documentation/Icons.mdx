# Темизация svg-пиктограмм

## Варианты вставки svg-пиктограмм:

1. Вставка из стандартного набора иконок. Необходимо передать наименование иконки в соответствующее поле.

<Source
  code={`<dx-button meButton leftIcon="arrowback"></dx-button>`}
  language="html"
  dark
/>

2. Вставка svg-пиктограммы строкой. Необходимо передать код иконки или переменную с кодом в соответствующее поле, например:

<Source
  code={`const cancelIcon = 
    \`<svg width="iconSize" height="iconSize" viewBox="0 0 16 16" ...>
      <path d="M5.0625 12L8 9.0625L10.9375 ..." fill="color"/>
    </svg>\`;`}
  language="js"
  dark
/>

Чтобы была возможность изменять размер и цвет иконки, необходимо в свойства <code>width</code> и <code>height</code> тега svg передать строку "iconSize",
а в свойство <code>fill</code> тега <code>path</code> строку "color".

<Source
  code={`<dx-button meButton leftIcon="cancelIcon"></dx-button>`}
  language="html"
  dark
/>

3. Вставка svg-пиктограмм, расположеных в папке <code>assets/images/icons/light</code> и в папке <code>assets/images/icons/dark</code>.
   Иконка для темной и светлой темы должна иметь одинаковое название. Это название передается в соответствующее свойство.

<Source
  code={`<dx-button meButton rightIconName="cached.svg"></dx-button>`}
  language="html"
  dark
/>

<b>
  Для каждого из вариантов вставки пиктограмм предусмотрена автоматическая смена
  цвета иконки при смене темы.
</b>

1. Для вставки из стандартного набора иконок:

   - при неуказанном значении цвета иконки, svg-пиктограмма принимает стандартные для компонента цвета из дизайн-системы и автоматически перекрашивается при изменении темы
   - при необходимости применить другой цвет, необходимо выбрать переменную из списка стандартных цветов, которые будут автоматически меняться при смене темы
     <b>
       Если установить кастомное значение цвета, при переключении темы будет
       применяться только это значение!
     </b>

2. Для вставки svg-пиктограммы строкой:

   - при неуказанном значении цвета иконки, svg-пиктограмма принимает стандартные для компонента цвета из дизайн-системы и автоматически перекрашивается при изменении темы
   - при необходимости применить другой цвет, необходимо выбрать переменную из списка стандартных цветов, которые будут автоматически меняться при смене темы
     <b>
       Если установить кастомное значение цвета, при переключении темы будет
       применяться только это значение!
     </b>

3. Для вставки пиктограммы из assets.

   Этот вариант предусмотрен для тех случаев, когда стандартные иконки или цвета не покрывают требований проекта.
   В этом случае необходимо подготовить две иконки, отдельно иконка для светлой темы, отдельно для темной.

   Готовые иконки поместить в папки:

   - <code>assets/images/icons/light</code> - иконку для светлой темы
   - <code>assets/images/icons/dark</code> - иконку для темной темы{" "}

   В этом случае при переключении темы, будет изменяться путь к иконке и соответственно меняться цвет
   иконки или ей вид. <b>Название иконок должно быть одинаковым.</b>
   Попытка передать в свойство принимающее цвет иконки какое-либо другое значение
   ни к какому результату не приведет.
