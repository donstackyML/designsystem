# Button

## Директива meButton

Применение директивы <code>meButton</code> к элементу <code>dx-button</code> приводит его размеры к значению <code>size = medium (32)</code> из дизайн-системы и предоставляет доступ к другим свойствам.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Angular/Documentation/ApiReference/UI_Components/dxButton/)

## Иконка слева

<code>leftIcon</code> принимает название иконки из стандартного набора, которая будет
вставлена слева от текста

<Story of={buttonStories.LeftIcon} />

<Source
  code={`<dx-button meButton text="Button" leftIcon="arrowback"></dx-button>`}
  language="html"
  dark
/>

## Иконка слева и справа

<code>rightIcon</code> принимает название иконки из стандартного набора, которая
будет вставлена справа от текста.

<Story of={buttonStories.LeftRightIcon} />

<Source
  code={`<dx-button meButton text="Button" leftIcon="arrowback" rightIcon="arrowforward"></dx-button>`}
  language="html"
  dark
/>

## Иконка без текста

Принимает иконку для кнопки только с иконкой, без текста

<Source code={`<dx-button meButton iconOnly="add"></dx-button>`} dark />
