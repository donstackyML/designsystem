# Accordion

## Директива meAccordion

Директива `meAccordion` применяется к элементу dx-accordion для приведения его размеров в соответствие с дизайн-системой и добавления дополнительных свойств.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Documentation/ApiReference/UI_Components/dxAccordion/)

## Изменение размера аккордеона

Для изменения размера аккордеона используйте параметр `size` с возможными значениями `small`, `medium`, или `large`.

<Story of={AccordionStories.Small} />

<Source
  code={`<dx-accordion meAccordion [size]="'small'"> <!-- Ваш контент здесь --> </dx-accordion>`}
  dark
  language='html'
/>

## Сворачиваемый аккордеон

Чтобы сделать аккордеон сворачиваемым, установите параметр `isCollapsible` в `true`.

<Story of={AccordionStories.Collapsible} />

<Source
  code={`<dx-accordion meAccordion [isCollapsible]="true"> <!-- Ваш контент здесь --> </dx-accordion>`}
  dark
  language='html'
/>

## Множественное раскрытие

Для возможности одновременного раскрытия нескольких элементов, установите параметр `isMultiple` в `true`.

<Story of={AccordionStories.Multiple} />

<Source
  code={`<dx-accordion meAccordion [isMultiple]="true"> <!-- Ваш контент здесь --> </dx-accordion>`}
  dark
  language='html'
/>

## Настройка анимации

Вы можете настроить длительность анимации с помощью параметра `animationDuration`.

<Story of={AccordionStories.CustomAnimation} />

<Source
  code={`<dx-accordion meAccordion [animationDuration]="1000"> <!-- Ваш контент здесь --> </dx-accordion>`}
  dark
  language='html'
/>

## Добавление пользовательского класса

Для добавления пользовательского класса используйте параметр `customClass`.

<Story of={AccordionStories.CustomClass} />

<Source
  code={`<dx-accordion meAccordion customClass="my-custom-accordion"> <!-- Ваш контент здесь --> </dx-accordion>`}
  dark
  language='html'
/>
