# Context Menu

## Директива meContextMenu

Применение директивы <code>meContextMenu</code> к элементу <code>dx-context-menu</code> приводит его внешний вид в соответствие с дизайн-системой.

## Добавление заголовка в контекстное меню

В DevExtreme для компонента Context Menu заголовок не предусмотрен.
Предлагается для добавления заголовка стилизовать один из пунктов меню с помощью шаблона <code>menu-title</code>. Например,

<Source
  code={`
  contextMenuItems: dxContextMenuItem[] = [
    {
      text: 'Заголовок',
      icon: this.iconService.getIcon({ icon: 'public', size: '24' }),
      template: 'menu-title',
      disabled: true,
    },
    // ...остальные элементы
    
`}
  dark
  language="js"
/>

Обратите внимание, для того чтобы отключить стандартные состояния <code>(hover, active)</code>
заголовку добавлено свойство <code>disabled</code> со значением <code>true</code>.

Классы <code>me-menu-title</code> и <code>me-title-header2</code> предоставляются библиотекой и
стилизуют расположение и вид элементов в соответствии с дизайн-системой.
