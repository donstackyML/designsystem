# Autocomplete

## Директива meAutocomplete

Директива `meAutocomplete` добавляет функциональность для компонента DevExtreme `DxAutocomplete`, позволяя настраивать размер и поведение скроллбара.

## Обработка событий

`meAutocomplete` поддерживает стандартные события компонента `DxAutocomplete`. Вот пример обработки событий изменения значения:

<Source
  code={`
<dx-autocomplete
  meAutocomplete
  [(value)]="value"
  (onValueChanged)="onValueChanged($event)"
></dx-autocomplete>

  // В вашем компоненте:
  onValueChanged(e: any) {
    console.log('Value changed:', e);
  }
`}
  dark
  language="typescript"
/>

### Использование различных размеров

Директива `meAutocomplete` позволяет настраивать различные размеры автокомплита:

- Маленький (`small`)
- Средний (`medium`)
- Большой (`large`)

<Source
  code={`
<dx-autocomplete meAutocomplete size="small"></dx-autocomplete>
<dx-autocomplete meAutocomplete size="medium"></dx-autocomplete>
<dx-autocomplete meAutocomplete size="large"></dx-autocomplete>
`}
  dark
  language="html"
/>

Это позволяет адаптировать компонент к различным требованиям интерфейса.

### Использование скроллбара

В зависимости от параметра `showScrollbar`, можно задать отображение скроллбара:

- Всегда (`always`)
- При наведении (`onHover`)
- Никогда (`never`)

<Source
  code={`
<dx-autocomplete meAutocomplete showScrollbar="always"></dx-autocomplete>
<dx-autocomplete meAutocomplete showScrollbar="onHover"></dx-autocomplete>
<dx-autocomplete meAutocomplete showScrollbar="never"></dx-autocomplete>
`}
  dark
  language="html"
/>

Таким образом, `meAutocomplete` предоставляет гибкие настройки для кастомизации поведения скроллбара.
