# ProgressBar

## Директива meProgressBar

Директива `meProgressBar` расширяет настройки и изменяет внешний вид стандартного компонента ProgressBar DevExtreme, добавляя дополнительные размеры.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Angular/Documentation/23_1/Guide/UI_Components/ProgressBar/)
