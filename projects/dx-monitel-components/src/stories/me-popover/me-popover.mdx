# Popover

Компонент `MePopoverDirective` предоставляет возможность создания всплывающих подсказок и диалогов.

<Story of={PopoverStories.Default} />

## Пример использования

```html
<dx-popover
  mePopover
  target="#popoverTarget"
  [showEvent]="showEvent"
  [hideEvent]="hideEvent"
  [position]="position"
  [width]="width"
  [maxWidth]="maxWidth"
  [showTitle]="showTitle"
  [title]="title"
  [shading]="shading"
  [shadingColor]="shadingColor"
  [size]="size"
>
  <div *dxTemplate="let data of 'content'">{{ content }}</div>
</dx-popover>
```

## Дополнительная информация

Свойства showEvent и hideEvent определяют, при каких действиях поповер будет показываться и скрываться.
Свойство position задает расположение поповера относительно целевого элемента.
