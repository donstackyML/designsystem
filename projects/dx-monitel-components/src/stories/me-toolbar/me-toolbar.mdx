# Toolbar

## Директива meToolbar

Применение директивы <code>meToolbar</code> к элементу <code>dx-toolbar</code> приводит его размеры к значению <code>size=medium</code> из дизайн-системы и предоставляет доступ к другим свойствам. Изменение размеров элементов тулбара происходит путем добавления соответствующих директив и установки нужных размеров.

## Добавление кнопок с иконками

Для добавления кнопок с иконками необходимо использовать компонент <code>dx-button</code> с директивой <code>meButton</code> и задать свойство <code>iconOnly</code> с названием иконки.

<Canvas of={toolbarStories.IconButtons} />

<Source
  code={`
<dx-toolbar meToolbar [size]="size">
  <dxi-item location="before">
    <div *dxTemplate>
      <dx-button meButton iconOnly="undo" [size]="size" iconColor="#18181a"></dx-button>
    </div>
  </dxi-item>
  <dxi-item location="before">
    <div *dxTemplate>
      <dx-button meButton iconOnly="redo" [size]="size" iconColor="#18181a"></dx-button>
    </div>
  </dxi-item>
  <!-- Другие элементы -->
</dx-toolbar>
`}
  language="html"
  dark
/>

## Добавление выпадающих кнопок

Для добавления выпадающих кнопок используйте компонент <code>dx-drop-down-button</code> с директивой <code>meDropDownButton</code>.

<Canvas of={toolbarStories.DropDownButtons} />

<Source
      code={`
    <dx-toolbar meToolbar [size]="size">
    <dxi-item location="before" locateInMenu="auto">
    <div *dxTemplate>
    <dx-drop-down-button
    meDropDownButton
    [size]="size"
    text="1.35"
    width="100%"
    displayExpr="text"
    keyExpr="value"
    [items]="lineHeights"
    stylingMode="contained"
    ></dx-drop-down-button>
    </div>
    </dxi-item>
      <!-- Другие элементы -->
    </dx-toolbar>
    `}
      language="html"
      dark
    />

    ## Изменение размера Toolbar

    Размер Toolbar можно изменить с помощью свойства <code>size</code>, доступного в директиве <code>meToolbar</code>. Доступны размеры <code>small</code>, <code>medium</code>, <code>large</code>.

    <Canvas of={toolbarStories.SizeLarge} />

<Source
  code={`
<dx-toolbar meToolbar size="large">
  <!-- Ваши элементы -->
</dx-toolbar>
`}
  language="html"
  dark
/>

## Добавление разделителей

Для добавления разделителей используйте класс <code>me-toolbar-separator</code> внутри элемента <code>dxi-item</code>.

<Canvas of={toolbarStories.Separators} />

<Source
  code={`
<dx-toolbar meToolbar [size]="size">
  <dxi-item location="before">
    <div *dxTemplate>
      <div class="me-toolbar-separator"></div>
    </div>
  </dxi-item>
  <!-- Другие элементы -->
</dx-toolbar>
`}
  language="html"
  dark
/>

## Использование кнопок с директивой meButton

Все кнопки внутри Toolbar должны использовать директиву <code>meButton</code> для соответствия дизайн-системе.

    <Canvas of={toolbarStories.MeButtonUsage} />

    <Source
      code={`
    <dx-toolbar meToolbar [size]="size">
    <dxi-item location="before">
    <div *dxTemplate>
    <dx-button meButton iconOnly="add" [size]="size" iconColor="#18181a"></dx-button>
    </div>
    </dxi-item>
      <!-- Другие элементы -->
    </dx-toolbar>
    `}
      language="html"
      dark
    />

    ## Добавление иконок из набора

    Вы можете использовать иконки из набора, предоставленного в вашем проекте. Для этого укажите название иконки в свойстве <code>iconOnly</code> компонента <code>dx-button</code> с директивой <code>meButton</code>.

    <Canvas of={toolbarStories.CustomIcons} />

    <Source
    code={`
    <dx-toolbar meToolbar [size]="size">
    <dxi-item location="before">
    <div *dxTemplate>
    <dx-button meButton iconOnly="format_bold" [size]="size" iconColor="#18181a"></dx-button>
    </div>
    </dxi-item>
    <dxi-item location="before">
    <div *dxTemplate>
    <dx-button meButton iconOnly="format_italic" [size]="size" iconColor="#18181a"></dx-button>
      </div>
    </dxi-item>
    <dxi-item location="before">
    <div *dxTemplate>
    <dx-button meButton iconOnly="format_underlined" [size]="size" iconColor="#18181a"></dx-button>
      </div>
    </dxi-item>
      <!-- Другие элементы -->
    </dx-toolbar>
    `}
      language="html"
      dark
    />
