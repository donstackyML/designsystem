# MeCard

## Компонент MeCard

Компонент `MeCard` представляет собой карточку с настраиваемыми размерами, заголовком и подвалом. В этом примере мы демонстрируем несколько вариантов использования компонента, включая различный размер и поведение при переполнении контента.

## Обработка событий

Карточка поддерживает обработку событий, такие как клики по кнопкам. Пример обработки событий в компоненте:

  <Source
  code={`
<me-card [size]="'medium'" [showHeader]="true" [showFooter]="true" [markerColor]="'#4CAF50'">
  <ng-container card-header-left>
    <me-icon icon="home" size="medium"></me-icon>
    <span>Card Title</span>
  </ng-container>
  <ng-container card-header-right>
    <dx-button iconOnly="add" (onClick)="onAddClick()"></dx-button>
    <dx-button iconOnly="arrowforward" (onClick)="onForwardClick()"></dx-button>
  </ng-container>
  <div>
    <p>Card content goes here. This is a basic card example.</p>
    <me-chips-container [chips]="[{ label: 'Example Chip', size: 'medium', count: 1 }]"></me-chips-container>
  </div>
  <ng-container card-footer>
    <dx-button text="Cancel" type="normal" stylingMode="outlined" (onClick)="onCancelClick()"></dx-button>
    <dx-button text="Save" type="success" (onClick)="onSaveClick()"></dx-button>
  </ng-container>
</me-card>

    // В вашем компоненте:
    onAddClick() {
      console.log('Add button clicked');
    }

    onForwardClick() {
      console.log('Forward button clicked');
    }

    onCancelClick() {
      console.log('Cancel button clicked');
    }

    onSaveClick() {
      console.log('Save button clicked');
    }

    onCloseClick() {
      console.log('Close button clicked');
    }
`}
  dark
  language="typescript"
/>
