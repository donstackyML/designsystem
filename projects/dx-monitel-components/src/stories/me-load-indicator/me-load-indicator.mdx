# meLoadIndicator

## Директива meLoadIndicator

Директива `meLoadIndicator` позволяет создавать индикаторы загрузки с различными параметрами, включая размер и цвет. Индикаторы могут быть настроены с помощью CSS классов и внешних ресурсов.

<Source of={loadIndicatorStories.Default} dark />

## Примеры использования:

- [Изменение размера индикатора](#изменение-размера-индикатора)
- [Изменение цвета индикатора](#изменение-цвета-индикатора)
- [Использование пользовательского индикатора](#использование-пользовательского-индикатора)

## Изменение размера индикатора

Для изменения размера индикатора используйте параметр `size`.

<Source
  code={`<div id="small-indicator" meLoadIndicator [size]="'small'" [color]="'normal'"></div>`}
  dark
  language="html"
/>

## Изменение цвета индикатора

Для изменения цвета индикатора используйте параметр `color`.

<Source
  code={`<div id="medium-indicator" meLoadIndicator [size]="'medium'" [color]="color"></div>`}
  dark
  language="html"
/>

## Использование пользовательского индикатора

Для использования пользовательского индикатора используйте параметр `indicatorSrc`.

<Source
  code={`<div id="image-indicator" meLoadIndicator [size]="'large'" [indicatorSrc]="indicatorSrc"></div>`}
  dark
  language="html"
/>
