# MeLoadPanel

## Директива MeLoadPanel

Директива `MeLoadPanel` позволяет создавать панели загрузки с различными параметрами, включая размер, отображение индикатора и затенение. Панели могут быть стилизованы с помощью CSS классов.

## Маленькая панель загрузки

Для создания маленькой панели загрузки используйте параметр `size` со значением `small`.

<Story of={loadPanelStories.Small} />

<Source
  code={`<dx-load-panel
  meLoadPanel
  [size]="'small'"
  shadingColor="rgba(0,0,0,0.4)"
  [position]="{ of: '#employee' }"
  [(visible)]="loadingVisible"
  [showIndicator]="showIndicator"
  [showPane]="showPane"
  [shading]="shading"
  [hideOnOutsideClick]="hideOnOutsideClick"
  (onShown)="onShown()"
  (onHidden)="onHidden()"
></dx-load-panel>`}
  dark
  language="html"
/>

## Панель с пользовательским классом

Для добавления пользовательского класса используйте параметр `customClass`.

<Story of={loadPanelStories.CustomClass} />

<Source
  code={`<dx-load-panel

  shadingColor="rgba(0,0,0,0.4)"
