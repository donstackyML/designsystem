# MeBreadcrumbs

MeBreadcrumbs компонент предоставляет навигационную цепочку для отображения текущего местоположения пользователя в иерархии сайта.

## Основные возможности

- Отображение иерархии навигации
- Поддержка иконок для каждого элемента
- Возможность добавления выпадающих меню
- Автоматическое сокращение элементов при нехватке места
- Настраиваемое направление сокращения (слева или справа)

### Сокращение слева

## Обработка событий

Компонент MeBreadcrumbs поддерживает событие `itemClick` для обработки кликов по элементам:

```html
<me-breadcrumbs
  [items]="breadcrumbItems"
  (itemClick)="onBreadcrumbItemClick($event)"
></me-breadcrumbs>
```
