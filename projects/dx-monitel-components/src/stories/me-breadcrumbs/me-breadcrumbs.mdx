# MeBreadcrumbs

## Компонент MeBreadcrumbs

Компонент `MeBreadcrumbs` предоставляет навигационную цепочку для отображения текущего местоположения пользователя в иерархии сайта. Он улучшает пользовательский опыт, позволяя легко ориентироваться по структуре приложения.

[Документация по компоненту MeBreadcrumbs](https://your-documentation-link.com/me-breadcrumbs)

## Сокращение слева

По умолчанию, при нехватке места элементы сокращаются справа. Вы можете изменить это поведение, установив `truncateFrom="left"`:

<Canvas of={BreadcrumbsStories.TruncateFromLeft} />
<Source
  code={`
<me-breadcrumbs
  [items]="breadcrumbItems"
  truncateFrom="left"
></me-breadcrumbs>
  `}
  dark
  language="html"
/>

## Дополнительная информация

При использовании компонента `MeBreadcrumbs` обратите внимание на следующие моменты:

- Свойство `size` может принимать значения `small`, `medium` или `large`, влияя на размер хлебных крошек.
- Свойство `truncateFrom` определяет, с какой стороны будут сокращаться элементы при нехватке места (`left` или `right`).
