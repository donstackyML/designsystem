# MeBreadcrumbs

MeBreadcrumbs - это компонент для создания навигационных цепочек ("хлебных крошек") в вашем приложении.

## Основные возможности

- Отображение иерархии навигации
- Поддержка иконок для элементов
- Возможность создания подменю для элементов
- Адаптивный дизайн с автоматическим скрытием элементов при недостатке места

### BreadcrumbItem

Интерфейс `BreadcrumbItem` описывает структуру элемента хлебных крошек:

```typescript
interface BreadcrumbItem {
  text: string;    // Текст элемента
  url?: string;    // URL для перехода (опционально)
  icon?: string;   // Иконка элемента (опционально)
  items?: BreadcrumbItem[]; // Подэлементы для создания подменю (опционально)
}
```

## События

- `itemClick`: Событие, возникающее при клике на элемент хлебных крошек. Передает объект `BreadcrumbItem`.

## Стилизация

Компонент использует DevExtreme Menu для отображения элементов, что позволяет легко кастомизировать внешний вид с помощью CSS. Основные классы для стилизации:

- `.breadcrumbs`: Контейнер для всех элементов
- `.breadcrumb-item`: Отдельный элемент хлебных крошек
- `.separator`: Разделитель между элементами

## Доступность

Компонент использует семантический тег `<nav>` с атрибутом `aria-label="Breadcrumbs"` для улучшения доступности.
