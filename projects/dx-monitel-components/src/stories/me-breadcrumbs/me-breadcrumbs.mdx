# Breadcrumbs

Компонент `MeBreadcrumbs` предоставляет навигационную цепочку для отображения текущего положения пользователя в иерархической структуре сайта.

## Особенности

- Отображение всех элементов навигационной цепочки
- Поддержка иконок для каждого элемента
- Использование DevExtreme кнопок для улучшенного пользовательского опыта
- Визуальное разделение элементов с помощью иконки стрелки
- Последний элемент (текущая страница) стилизован как неактивный

## Использование

````tsx
<me-breadcrumbs
  [items]="breadcrumbItems"
  (itemClick)="onItemClick($event)"
></me-breadcrumbs>```
````
