# TreeView

## Директива meTreeView

Применение директивы `meTreeView` к элементу `dx-tree-view` приводит его внешний вид в соответствие с дизайн-системой, его размеры к значению `size = large (24)` и предоставляет доступ к свойству `size`.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Angular/Documentation/ApiReference/UI_Components/dxTreeView/)

[Ссылка на Demo компонента DevExtreme](https://js.devexpress.com/Angular/Demos/WidgetsGallery/Demo/TreeView/Basics/MaterialBlueLight/)
