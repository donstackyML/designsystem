# TreeView

## Директива meTreeView

Применение директивы <code>meTreeView</code> к элементу <code>dx-tree-view</code> приводит его внешний вид в соответствие
с дизайн-системой, его размеры к значению <code>size = large (24)</code> и предоставляет доступ к свойству <code>size</code>.
