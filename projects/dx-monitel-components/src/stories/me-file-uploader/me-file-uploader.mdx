# FileUploader

## Директива meFileUploader

Директива `meFileUploader` применяется к элементу dx-file-uploader для управления его состояниями и добавления дополнительных свойств.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Documentation/ApiReference/UI_Components/dxFileUploader/)

## Отключенный файл загрузчика

Чтобы сделать файл загрузчика отключенным, установите параметр `isDisabled` в `true`.

<Story of={FileUploaderStories.Disabled} />

<Source
  code={`<dx-file-uploader meFileUploader [isDisabled]="true"> <!-- Ваш контент здесь --> </dx-file-uploader>`}
  dark
  language='html'
/>

## Кастомный класс

Для добавления пользовательского класса используйте параметр `customClass`.

<Story of={FileUploaderStories.CustomClass} />

<Source
  code={`<dx-file-uploader meFileUploader customClass="my-custom-class"> <!-- Ваш контент здесь --> </dx-file-uploader>`}
  dark
  language='html'
/>

## Множественные файлы

Для возможности загрузки нескольких файлов установите параметр `multiple` в `true`.

<Story of={FileUploaderStories.MultipleFiles} />

<Source
  code={`<dx-file-uploader meFileUploader [multiple]="true"> <!-- Ваш контент здесь --> </dx-file-uploader>`}
  dark
  language='html'
/>
