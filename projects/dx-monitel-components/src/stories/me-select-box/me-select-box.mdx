# SelectBox

## Директива meSelectBox, meLabel

Применение директивы <code>meSelectBox</code> к элементу <code>dx-select-box</code> приводит его размеры
к значению <code>size=medium (32)</code> из дизайн-системы, задает по-умолчанию <code>stylingMode=filled</code> и предоставляет доступ к другим свойствам.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Angular/Documentation/ApiReference/UI_Components/dxSelectBox/)

## Определение положения лейбла снаружи текстового поля

Применение директивы <code>meLabel</code> позволяет определить положение лейбла снаружи текстового поля (сверху или слева).

<code>labelDirection</code> определяет положение лейбла снаружи текстового поля (сверху
или слева).

### Положение лейбла сверху

<Story of={selectBoxStories.WithLabelColumn} />

<Source
  code={`<label meLabel labelDirection="column">
    Label
    <dx-select-box meSelectBox [dataSource]="dataSource"></dx-select-box>
</label>`}
  dark
/>

### Положение лейбла слева

  <Story of={selectBoxStories.WithLabelRow} />

  <Source
    code={`<label meLabel labelDirection="row" width="250px">
    Label
    <dx-select-box meSelectBox [dataSource]="dataSource"></dx-select-box>
    </label>`}
      dark
    />

    <code>width</code> определяет размер текстового поля вместе с лейблом (используется
    в теге <code>label</code> для расположения <code>labelPosition="left"</code>)
