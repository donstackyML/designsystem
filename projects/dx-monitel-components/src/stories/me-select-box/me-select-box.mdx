# SelectBox

## Директивы meSelectBox, meLabel

Применение директивы <code>meSelectBox</code> к элементу <code>dx-select-box</code> приводит его размеры к значению <code>size=medium (32)</code> из дизайн-системы, задает по умолчанию <code>stylingMode=outlined</code> и предоставляет доступ к другим свойствам.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Angular/Documentation/ApiReference/UI_Components/dxSelectBox/)

## Свойства компонента

| Свойство       | Тип              | Описание                                                       |
| -------------- | ---------------- | -------------------------------------------------------------- |
| dataSource     | any              | Задает или получает массив данных, отображаемых компонентом.   |
| placeholder    | string           | Задает текст, отображаемый, когда компонент не имеет значения. |
| searchEnabled  | boolean          | Определяет, разрешен ли поиск в списке элементов.              |
| disabled       | boolean          | Определяет, отключен ли компонент.                             |
| readOnly       | boolean          | Определяет, доступен ли компонент только для чтения.           |
| stylingMode    | string           | Задает режим стилизации компонента.                            |
| width          | number \| string | Задает ширину компонента.                                      |
| label          | string           | Задает текст метки для компонента.                             |
| labelMode      | string           | Определяет режим отображения метки.                            |
| size           | string           | Задает размер компонента.                                      |
| isValid        | boolean          | Определяет, является ли текущее значение допустимым.           |
| showClearButton| boolean          | Показывает или скрывает кнопку очистки.                        |

## Определение положения лейбла снаружи SelectBox

Применение директивы <code>meLabel</code> позволяет определить положение лейбла снаружи SelectBox (сверху или слева).

<code>labelDirection</code> определяет положение лейбла снаружи SelectBox (сверху или слева).

### Положение лейбла сверху

<Canvas of={selectBoxStories.WithLabelColumn} />

<Source
  code={`
<label meLabel labelDirection="column">
  Выберите продукт
  <dx-select-box meSelectBox [dataSource]="dataSource"></dx-select-box>
</label>
`}
  dark
  language="html"
/>

### Положение лейбла слева

    <Canvas of={selectBoxStories.WithLabelRow} />

    <Source
      code={`
    <label meLabel labelDirection="row" width="250px">
    Выберите продукт
    <dx-select-box meSelectBox [dataSource]="dataSource"></dx-select-box>
    </label>
    `}
      dark
      language="html"
    />

    <code>width</code> определяет размер SelectBox вместе с лейблом (используется в теге <code>label</code> для расположения <code>labelDirection="row"</code>)
