# SelectBox

## Директива meSelectBox, meLabel

Применение директивы <code>meSelectBox</code> к элементу <code>dx-select-box</code>
приводит его размеры к значению <code>size=medium (32)</code> из дизайн-системы, задает по умолчанию <code>stylingMode=filled</code> и предоставляет доступ к другим свойствам.

## Определение положения лейбла снаружи SelectBox

Применение директивы <code>meLabel</code> позволяет определить положение лейбла снаружи SelectBox (сверху или слева).

<code>labelDirection</code> определяет положение лейбла снаружи SelectBox (сверху
или слева).

### Положение лейбла сверху

Для положения `label` сверху используйте тэг <code>label</code> с директивой <code>meLabel</code> и установите параметр <code>labelDirection="column"</code>. Также установите необходимые стили выравнивания как <code>align-items: flex-start</code>.
