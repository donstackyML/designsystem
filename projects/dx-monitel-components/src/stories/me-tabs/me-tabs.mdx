# Tabs

## Директива meTabs

Директива `meTabs`, применяемая к компоненту `<dx-tabs>`, стилизует его в соответствии с дизайн-системой, добавляет дополнительные свойства и предоставляет возможность гибкой настройки внешнего вида вкладок.

[Документация по компоненту DevExtreme Tabs](https://js.devexpress.com/Documentation/ApiReference/UI_Components/dxTabs/)

## Вертикальная ориентация

Для отображения вкладок в вертикальной ориентации используйте свойство `orientation="vertical"`.

<Canvas of={tabsStories.VerticalOrientation} />

<Source
  code={`
<dx-tabs
  meTabs
  [orientation]="vertical"
  [dataSource]="tabsData"
></dx-tabs>
`}
  dark
  language="html"
/>

## С кнопками навигации

Чтобы добавить кнопки навигации при наличии большого количества вкладок, используйте свойство `showNavButtons="true"` и задайте фиксированную ширину компонента.

<Canvas of={tabsStories.WithNavButtons} />

<Source
  code={`
<dx-tabs
  meTabs
  [showNavButtons]="true"
  width="300px"
  [dataSource]="tabsData"
></dx-tabs>
`}
  dark
  language="html"
/>

### rtlEnabled

Задает поддержку отображения элемента управления для языков с написанием справа налево.

- Тип: `boolean`
- Значение по умолчанию: `false`

**Описание:**

Когда свойство `rtlEnabled` установлено в `true`, компоненты DevExtreme отображают свой контент в направлении, соответствующем языкам с письмом справа налево (Right-To-Left). Это влияет на расположение текста, иконок и других элементов интерфейса.

**Пример использования:**

```html
<dx-tabs
  meTabs
  [rtlEnabled]="true"
  [dataSource]="tabsData"
  [selectedIndex]="0"
></dx-tabs>
`} dark language="html" /> ## Дополнительная информация При использовании
```
