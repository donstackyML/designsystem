# Tabs

## Директива meTabs

Директива `meTabs`, применяемая к компоненту `<dx-tabs>`, стилизует его в соответствии с дизайн-системой, добавляет дополнительные свойства и предоставляет возможность гибкой настройки внешнего вида вкладок.

[Документация по компоненту DevExtreme Tabs](https://js.devexpress.com/Documentation/ApiReference/UI_Components/dxTabs/)

## Вертикальная ориентация

Для отображения вкладок в вертикальной ориентации используйте свойство `orientation="vertical"`.

<Canvas of={tabsStories.VerticalOrientation} />

<Source
  code={`
<dx-tabs
  meTabs
  orientation="vertical"
  [dataSource]="tabsData"
></dx-tabs>
`}
  dark
  language="html"
/>

## С кнопками навигации

Чтобы добавить кнопки навигации при наличии большого количества вкладок, используйте свойство `showNavButtons="true"` и задайте фиксированную ширину компонента.

<Canvas of={tabsStories.WithNavButtons} />

<Source
  code={`
<dx-tabs
  meTabs
  [showNavButtons]="true"
  width="300px"
  [dataSource]="tabsData"
></dx-tabs>
`}
  dark
  language="html"
/>

## Прокрутка содержимого

Для включения прокрутки вкладок при большом количестве используйте свойство `scrollByContent="true"`.

## Дополнительная информация

При использовании компонента `<dx-tabs>` с директивой `meTabs` обратите внимание на следующие моменты:

- Свойство `size` может принимать значения `small`, `medium` или `large`, влияя на размер вкладок.
- Свойство `orientation` определяет ориентацию вкладок: `horizontal` или `vertical`.
- Свойство `stylingMode` позволяет переключаться между стилями оформления: `primary` и `secondary`.
- Свойство `iconPosition` задает позицию иконки относительно текста: `top`, `start`, `end`, `bottom`.
