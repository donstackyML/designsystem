## Обзор

Свойство <code>isSidePageOpen</code> отвечает за показ <code>sidepage</code> и должно иметь двустороннее связывание со
свойством родительского компонента, которым планируется упралять появлением <code>sidepage</code>.

Родительский элемент для компонентов, которые требуется поместить в шапке <code>sidepage</code>, должен иметь аттрибут <code>sidepage-header</code>.
Родительский элемент для компонентов, которые требуется поместить в контентной части <code>sidepage</code>, должен иметь аттрибут <code>sidepage-content</code>
Родительский элемент для компонентов, которые требуется поместить в футере <code>sidepage</code>, должен иметь аттрибут <code>sidepage-footer</code>.

Классы <code>me-sidepage-header</code>, <code>me-sidepage-title</code>, <code>me-title-header1</code>,

<code>me-text-body2</code>, <code>me-text-body1</code>, <code>
  me-sidepage-footer
</code> встроены в библиотеку и позволяют стилизовать элементы <code>
  idepage
</code> в соответствии с дизайн-системой.
