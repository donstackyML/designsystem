# meToast

## Директива meToast

Директива `meToast` позволяет создавать уведомления с различными параметрами, включая время отображения, позицию и анимацию. Уведомления могут быть стилизованы с помощью CSS классов.

<Story of={toastStories.Default} />
<Controls of={toastStories.Default} />
<Source
  code={`<dx-toast 
  [message]="'This is a toast message'"
  [displayTime]="2000"
  [position]="'bottom right'"
  [animation]="{
    show: {type: 'fade', duration: 400, from: 0, to: 1},
    hide: {type: 'fade', duration: 400, from: 1, to: 0}
    }"
  [customClass]="''">
</dx-toast>`}
  dark
  language="html"
/>

## Изменение времени отображения

Для изменения времени отображения используйте параметр `displayTime`.

<Story of={toastStories.LongDuration} />

<Source
  code={`<dx-toast [displayTime]="5000" message="This toast will be displayed for 5 seconds"></dx-toast>`}
  dark
  language="html"
/>

## Изменение позиции уведомления

Для изменения позиции уведомления используйте параметр `position` с возможными значениями `top left`, `top center`, `top right`, `bottom left`, `bottom center`, `bottom right`.

<Story of={toastStories.CustomPosition} />

<Source
  code={`<dx-toast position="top center" message="This toast is positioned at the top center"></dx-toast>`}
  dark
  language="html"
/>

## Настройка анимации

Для настройки анимации используйте параметр `animation`.

<Story of={toastStories.CustomAnimation} />

<Source
  code={`<dx-toast
  [animation]="{
    show: { type: 'pop', duration: 400, from: { scale: 0 }, to: { scale: 1 } },
    hide: { type: 'pop', duration: 400, to: { scale: 0 } }
  }"
  message="This toast has a custom animation"
></dx-toast>`}
  dark
  language="html"
/>

## Применение пользовательского CSS класса

Для применения пользовательского CSS класса используйте параметр `customClass`.

<Story of={toastStories.CustomClass} />

<Source
  code={`<dx-toast customClass="my-custom-toast" message="This toast has a custom CSS class"></dx-toast>`}
  dark
  language="html"
/>
