# Tooltip

## Директива MeTooltip

Директива `MeTooltip` позволяет создавать информативные всплывающие подсказки для элементов пользовательского интерфейса.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Documentation/ApiReference/UI_Components/dxTooltip/)

## Позиционирование

Тултип можно расположить в различных позициях относительно целевого элемента.

<Story of={tooltipStories.Positioning} />

<Source
  code={`<button [meTooltip]="meTooltip" [tooltipPosition]="tooltipPosition">Тултип {{ tooltipPosition }}</button>`}
  dark
  language="html"
/>

## Анимация

Тултипы могут иметь анимацию появления и исчезновения.

{" "}

<Source
  code={`

## Кастомный шаблон

Вы можете использовать собственный шаблон для содержимого тултипа.

<Story of={tooltipStories.CustomTemplate} />

<Source
      code={`
<button [meTooltip]="tooltipTemplate" [tooltipTemplateRef]="tooltipTemplate" [tooltipPosition]="tooltipPosition">ултип с кастомным шаблоном</button>

<ng-template #tooltipTemplate>

   <div>
      <h4>Заголовок тултипа</h4>
      <p>Это пример тултипа с кастомной разметкой.</p>
      <ul>
      <li>Пункт 1</li>
      <li>Пункт 2</li>
      </ul>
    </div>
</ng-template>`}
dark
