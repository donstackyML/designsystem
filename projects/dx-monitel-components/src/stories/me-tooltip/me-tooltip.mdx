# Tooltip

## Директива MeTooltip

Директива `MeTooltip` позволяет создавать информативные всплывающие подсказки для элементов пользовательского интерфейса.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Documentation/ApiReference/UI_Components/dxTooltip/)

<Story of={tooltipStories.Basic} />

<Controls of={tooltipStories.Basic} />

## Примеры использования:

### Позиционирование

Тултип можно расположить в различных позициях относительно целевого элемента.

<Story of={tooltipStories.Positioning} />

<Source
  code={`<button [meTooltip]="meTooltip" [tooltipPosition]="tooltipPosition">Тултип {{ tooltipPosition }}</button>`}
  dark
  language="html"
/>

### HTML-содержимое

Тултипы могут содержать форматированный HTML-текст.

<Story of={tooltipStories.HTMLContent} />

<Source
  code={`<button [meTooltip]="'<strong>Жирный текст</strong> и <em>курсив</em>'" [tooltipPosition]="tooltipPosition">HTML-тултип</button>`}
  dark
  language="html"
/>

### Кастомные стили

Вы можете применить собственные стили к тултипу.

<Story of={tooltipStories.CustomStyles} />

<Source
  code={`<button [meTooltip]="meTooltip" [tooltipPosition]="tooltipPosition" [tooltipClass]="tooltipClass">Кастомный тултип</button>`}
  dark
  language="html"
/>

### Анимация

Тултипы могут иметь анимацию появления и исчезновения.

<Story of={tooltipStories.WithAnimation} />

<Source
  code={`
<button
  [meTooltip]="meTooltip"
  [tooltipPosition]="tooltipPosition"
  [tooltipShowAnimation]="tooltipShowAnimation"
  [tooltipHideAnimation]="tooltipHideAnimation">
  Анимированный тултип
</button>
  `}
  dark
  language="html"
/>

<Story of={tooltipStories.CustomTemplate} />

<Source
  code={`
<button [meTooltip]="tooltipTemplate"
        [tooltipPosition]="tooltipPosition"
        [tooltipTemplateRef]="tooltipTemplate">
  Тултип с кастомным шаблоном
</button>

  <!-- В компоненте: -->
  tooltipTemplate = \`
    <div>
      <h4>Заголовок тултипа</h4>
      <p>Это пример тултипа с кастомной разметкой.</p>
      <ul>
        <li>Пункт 1</li>
        <li>Пункт 2</li>
      </ul>
    </div>
  \`;
`}
  dark
  language="html"
/>
### Тултип с фиксированными размерами

Вы можете задать фиксированные размеры для тултипа.

<Story of={tooltipStories.CustomSizedTooltip} />

<Source
  code={`
<button [meTooltip]="meTooltip"
  [tooltipPosition]="tooltipPosition"
  [tooltipWidth]="tooltipWidth"
  [tooltipHeight]="tooltipHeight"
  [tooltipMaxWidth]="tooltipMaxWidth"
  [tooltipMaxHeight]="tooltipMaxHeight">
  Тултип с фиксированными размерами
  </button>
  `}
  dark
  language="html"
/>

### Тултип в контейнере

Вы можете ограничить область отображения тултипа определенным контейнером.

  <Story of={tooltipStories.ContainerTooltip} />

  <Source
    code={`
  <div id="tooltipContainer" style="position: relative; height: 100px; border: 1px solid #ccc; padding: 10px;">
  <button [meTooltip]="meTooltip"
  [tooltipPosition]="tooltipPosition">
  Тултип в контейнере
  </button>
  </div>
  `}
  dark
  language="html"
/>
