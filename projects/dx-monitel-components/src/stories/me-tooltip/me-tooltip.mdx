# Tooltip

## Директива Tooltip

Директива `Tooltip` позволяет создавать тултипы с различными параметрами, включая позиционирование, анимацию и пользовательские стили. Тултипы могут содержать текст, HTML и интерактивные элементы.

## Кастомные позиции

Для изменения позиции тултипа используйте параметр `tooltipPosition`.

<Story of={tooltipStories.CustomPositions} />

<Source
  code={`<button
  class="tooltip-button"
  [meTooltip]="simpleTooltip"
  [tooltipPosition]="simplePosition"
>
  {{ simpleButtonText }}
</button>`}
  dark
  language="html"
/>

## Кастомный контент

Для изменения контента тултипа используйте параметры `meTooltip` и `tooltipTemplateRef`.

<Story of={tooltipStories.CustomContent} />

<Source
  code={`<button
  class="tooltip-button"
  [meTooltip]="animatedTooltip"
  [tooltipPosition]="animatedPosition"
  [tooltipShowAnimation]="showAnimation"
  [tooltipHideAnimation]="hideAnimation"
>
  {{ animatedButtonText }}
</button>`}
  dark
  language="html"
/>

## Кастомные анимации

Для изменения анимаций тултипа используйте параметры `tooltipShowAnimation` и `tooltipHideAnimation`.

<Story of={tooltipStories.CustomAnimations} />

<Source
  code={`<button
  class="tooltip-button"
  [meTooltip]="positionedTooltip"
  [tooltipPosition]="positionedPosition"
  [tooltipShowAnimation]="showAnimation"
  [tooltipHideAnimation]="hideAnimation"
>
  {{ positionedButtonText }}
</button>`}
  dark
  language="html"
/>
