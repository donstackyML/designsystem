# CheckBox

## Директива meCheckBox

Применение директивы <code>meCheckBox</code> к элементу <code>dx-check-box</code> приводит его размеры к значению <code>size=medium (20)</code>
из дизайн-системы и предоставляет доступ к свойству <code>size</code>.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Angular/Documentation/ApiReference/UI_Components/dxCheckBox/)
