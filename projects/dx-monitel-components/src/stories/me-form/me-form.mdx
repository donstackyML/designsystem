# Form

Компонент `Form` предоставляет гибкий и настраиваемый интерфейс для создания форм в вашем приложении.

## Настройки

<Controls of={FormStories.Default} />

Компонент Form поддерживает различные настройки для кастомизации внешнего вида и поведения формы:

- `size`: Определяет размер формы ('small', 'medium', 'large').
- `labelMode`: Определяет, как отображаются метки полей (снаружи, статично, плавающие или скрытые).
- `labelLocation`: Устанавливает расположение меток (слева или сверху от полей).
- `colCount`: Задает количество колонок в форме.
- `readOnly`: Делает форму доступной только для чтения.
- `showColonAfterLabel`: Показывает или скрывает двоеточие после меток полей.

## Использование

Для использования компонента Form в вашем приложении, импортируйте `MeFormDirective` и добавьте его к декларациям вашего модуля:

```typescript
import { MeFormDirective } from 'path-to-me-form-directive';

@NgModule({
  // ...
  declarations: [MeFormDirective],
  // ...
})
export class YourModule { }
