# RadioGroup

## Директива meRadioGroup

Применение директивы <code>meRadioGroup</code> к элементу <code>dx-radio-group</code> приводит его
размеры к значению <code>size=medium (20)</code> из дизайн-системы и предоставляет доступ
к свойству <code>size</code>.

[Ссылка на API компонента DevExtreme](https://js.devexpress.com/Angular/Documentation/ApiReference/UI_Components/dxRadioGroup/)
