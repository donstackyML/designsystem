# MeTabPanel

## Директива MeTabPanel

Директива `MeTabPanel` позволяет создавать панели вкладок с различными параметрами, включая высоту, выбранный индекс, анимацию и возможность прокрутки. Директива также позволяет добавлять пользовательские классы CSS для стилизации.

<Canvas of={tabPanelStories.Default} />
<Controls of={tabPanelStories.Default} />
<Source of={tabPanelStories.Default} dark />

## С пользовательским классом

Для добавления пользовательского класса используйте параметр `customClass`.

<Canvas of={tabPanelStories.WithCustomClass} />

<Source
  code={`<dx-tab-panel meTabPanel [height]="'300px'" width="400px" [selectedIndex]="0" [animationEnabled]="true" [swipeEnabled]="true" customClass="custom-class"></dx-tab-panel>`}
  dark
  language="html"
/>

## Без анимации

Для отключения анимации используйте параметр `animationEnabled`.

<Canvas of={tabPanelStories.WithoutAnimation} />

<Source
  code={`<dx-tab-panel meTabPanel [height]="'300px'" width="400px" [selectedIndex]="0" [animationEnabled]="false" [swipeEnabled]="true"></dx-tab-panel>`}
  dark
  language="html"
/>

## Без возможности прокрутки

Для отключения возможности прокрутки используйте параметр `swipeEnabled`.

<Canvas of={tabPanelStories.WithoutSwipe} />

<Source
  code={`<dx-tab-panel meTabPanel [height]="'300px'" width="400px" [selectedIndex]="0" [animationEnabled]="true" [swipeEnabled]="false"></dx-tab-panel>`}
  dark
  language="html"
/>

## С пользовательской высотой

Для изменения высоты используйте параметр `height`.

<Canvas of={tabPanelStories.CustomHeight} />

<Source
  code={`<dx-tab-panel meTabPanel [height]="'500px'" width="400px" [selectedIndex]="0" [animationEnabled]="true" [swipeEnabled]="true"></dx-tab-panel>`}
  dark
  language="html"
/>
